About Ephemeral Notes

Ephemeral Notes is a lightweight, self-hostable web application for creating short-lived, unguessable notes intended for secure, ephemeral sharing.

Features

  • Create notes that automatically expire after a configurable time-to-live (TTL).
  • Optional password protection for viewing a note.
  • Optional Markdown formatting with safe sanitization to prevent XSS.
  • Optional "burn after view" that deletes a note on first successful view.
  • Lightweight creation history and total counts (metadata only â€” note contents are not stored in history).

Security and Privacy

Notes are stored in Redis with a time-to-live and are not retained beyond their configured expiration. When the "burn after view" option is used, the note is removed immediately after a successful view (server-side). Passwords (if used) are verified on the server and are not stored in plaintext.
